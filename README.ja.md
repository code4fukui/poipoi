# poipoi

福井県鯖江市における適切なごみの分別を促進し、楽しく学べるようにゲーミフィケーションを取り入れたWebアプリケーションです。ユーザーは分別のルールを検索したり、ごみの写真をアップロードしたり、クイズに参加したりできます。

## デモ

- **ライブアプリ:** [https://poipoi.life/](https://poipoi.life/)
- **VR180動画:** [static/movie/index.html](static/movie/index.html)

## 機能

- **ごみの分別ガイド:** 市の公式データに基づき、さまざまな品目の処分方法を検索できるインタラクティブなリスト。
- **写真アップロード:** 分別するごみの写真をアップロードできます。成功するとお祝いの音と紙吹雪のアニメーションが再生されます。
- **タイムラインとクイズ:** 最近投稿されたごみのタイムラインを表示し、各ごみの分別クイズで知識を試すことができます。
- **ゲーミフィケーション:** 「マイページ」では、ユーザーのモチベーションを高めるための「ポイ活（ポイント活動）」の模擬ポイントが表示されます。
- **市のダッシュボード:** 市の月間資源収入やごみ収集車の位置マップを示す模擬ダッシュボード。

## 技術スタック

- **ランタイム:** [Deno](https://deno.land/)
- **バックエンド:** Vanilla Denoと `js.sabae.cc` のユーティリティモジュールで構築されたシンプルなAPIサーバー。
- **フロントエンド:** Vanilla JavaScript (ES Modules)、HTML5、CSS。
- **データベース:** 投稿されたごみのデータを保存するフラットファイルJSONデータベース。

## 使い方

1. **Denoのインストール:** [Deno公式サイト](https://deno.land/#installation)の手順に従ってください。
2. **アプリケーションの実行:**
    ```sh
    deno run -A poipoi.js 8000
    ```
3. Webブラウザで [http://localhost:8000](http://localhost:8000) を開きます。

## データソース

このアプリケーションは、鯖江市が提供するオープンデータを使用しています。

- [ゴミの分別(福井県鯖江市) - データセット - オープンデータプラットフォーム | データカタログサイト](https://ckan.odp.jig.jp/dataset/jp-fukui-sabae-203-odp)
